== Description ==

The Modal Posts Images plugin adds modal behaviour to all of the wordpress image under post entry-content class. The plugin call all the images under 
entry-content class and add funtionality so that if it's clicked the post image will popup with a enlaged view. along with the image alt tag if it's 
present. Just installing and activating the plugin do the job.

Note: To get best experienced the image into the posts need to be implemented on full size.

Usage :

1. Download and extract this plugin to `wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress.
You will find the plugin activity on your wordpress posts page images.
